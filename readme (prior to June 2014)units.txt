Overview:
Extracts provide CIMIS weather data to the public via 
text files in "comma separated values" (CSV) format. 
Extract data may be imported into spreadsheet programs 
like Excel or into databases like Access or Oracle.

Historical Data prior to June 2014 will be found in the annual 
and annualMetric folders. Each extract file contains a year's 
worth of data for each station active during the noted year. 

Daily and Hourly data extract files are named using the
following pattern:

     YEARdailyXXX.csv
     YEARhourlyXXX.csv

The "XXX" represents the station number the file 
provides data for. Please refer to our public website 
for information on what station number would be of 
interest to you. You will likely need to know what 
County, Region, or Zip codes a station represents. 
This data can be easily obtained from the website: 

     www.cimis.water.ca.gov

Look under "STATIONS" --> "Station Location Map"

Data values will be provided in English and Metric units. 
Again, visit our website for further details.
